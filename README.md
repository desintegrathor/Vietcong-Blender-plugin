# Vietcong Blender Plugin

Import/export plugin for Vietcong BES model files (Blender 4.x).

## Features

- BES model import and export
- Standard, PteroMat, and PteroLayer material support
- Object hierarchy and LOD levels
- STG animation import/export (work in progress)

## Installation

1. Pack the `vietcong_bes/` folder into a `.zip` — the zip must contain the folder itself (not just its contents)
2. Open Blender and drag-and-drop the `.zip` onto the Blender window

## Usage

- **Import**: File > Import > Vietcong BES (.bes)
- **Export**: File > Export > Vietcong BES (.bes)
